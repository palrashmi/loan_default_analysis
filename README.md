
# Loan Default Analysis using Power BI

## Project Overview
This project analyzes loan default trends using Power BI, DAX, Dataflows, and interactive dashboards to identify high-risk borrowers and support better financial decision-making.

## Technologies Used
- Power BI Desktop
- Power BI Service
- Power Query
- DAX
- Excel
- Dataflow Gen1

---

## Key Features
- Interactive Power BI Dashboard
- Loan Default Risk Analysis
- Credit Score & DTI Analysis
- Year-over-Year Trend Analysis
- Dataflow-based ETL
- Scheduled Refresh in Power BI Service

---

## Key Insights
- Low credit score borrowers show higher default risk
- High DTI ratio strongly impacts loan defaults
- Employment type affects repayment behavior
- Young borrowers have higher default probability
